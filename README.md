<h2>Проекты</h2>

<p>
Примеры текущих разработок и аналитических инструментов для Bitrix24. Список будет регулярно пополняться новыми решениями:
</p>

<ul>
<li><b>Контроль трудозатрат и загрузки команды</b> — прозрачная аналитика по задачам сотрудников, выявление перерасхода времени и проблемных задач.</li>
<br>
<li><b>Аналитика продаж и прибыли по компаниям</b> — суммарные показатели сделок, прибыль и эффективность работы с клиентами за выбранный период.</li>
<br>
<li><b>Рейтинг сотрудников отдела продаж</b> — сравнительная таблица показателей по продажам, рост или снижение результатов, наглядная визуализация.</li>
<br>
<li><b>Контроль качества данных</b> — поиск проблемных сущностей, где отсутствуют ключевые поля, с возможностью фильтрации по типу и периоду.</li>
<br>
<li><b>Другие проекты и эксперименты</b> — новые отчёты и аналитические инструменты в разработке.</li>
</ul>

<p>
Каждый проект демонстрирует подход к решению прикладных задач и работе с данными.
</p>
